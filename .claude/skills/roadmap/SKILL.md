# /roadmap — 유형별 학습 로드맵 스킬

알고리즘 유형을 선택하면 난이도 순 학습 경로를 제시합니다.

## 입력 파싱

- `/roadmap dp` — DP 로드맵
- `/roadmap 그래프` — 그래프 로드맵
- `/roadmap dp 입문` — 난이도 지정 (입문/중급/고급)
- `/roadmap` — 인자 없음 → 사용자 현재 수준/관심사 질문

## 로드맵 구조

하나의 유형을 **3단계(입문 → 중급 → 고급)** 로 나눠 설명합니다.

각 단계마다:
- **학습 목표 1~2줄** — 이 단계에서 무엇을 체득해야 하는가
- **핵심 개념 체크리스트** — 이 단계 넘어가기 전에 알아야 할 것들
- **추천 문제 3~5개** — 각 문제에 한 줄 학습 포인트
- **다음 단계 진입 조건** — 언제 다음 단계로 넘어가도 되는가

## 답변 형식 예시

```
# DP 로드맵

## 입문 (1차원 DP, 점화식 세우기 연습)
목표: "상태 정의 → 점화식 → 초기값" 흐름에 익숙해지기

체크리스트:
- dp[i]가 무엇을 의미하는지 말로 설명할 수 있는가
- bottom-up과 top-down 차이를 아는가

추천 문제:
1. 1003 피보나치 함수 — 가장 기본적인 메모이제이션
2. 9095 1,2,3 더하기 — 경우의 수 점화식 감각
3. 11726 2xn 타일링 — 기하 문제를 DP로 바꾸는 첫 경험
4. 1463 1로 만들기 — 상태 전이가 여러 개인 경우

다음 단계 진입 조건: 위 4문제를 힌트 없이 풀 수 있으면 OK.

## 중급 (2차원 DP, 배낭)
...
```

## 주요 유형별 구성 요소 (참고)

### DP
- 입문: 1차원 DP (1463, 1003, 9095, 11726, 1149)
- 중급: 2차원 DP, 배낭 (12865, 9251 LCS, 11053 LIS, 2579 계단)
- 고급: 트리 DP, 비트마스크 DP, 구간 DP (2098 외판원, 11049 행렬 곱셈)

### 그래프
- 입문: BFS/DFS (1260, 2178, 7576)
- 중급: 다익스트라, 유니온파인드 (1753, 1717, 1197 MST)
- 고급: SCC, 이분매칭, 플로우 (2150, 2188)

### 자료구조
- 입문: 스택/큐/덱 (10828, 10845, 10866)
- 중급: 우선순위 큐, 유니온파인드 (1927, 1717)
- 고급: 세그먼트 트리, 펜윅 (2042, 2357, 11658)

### 이분탐색
- 입문: 기본 이분탐색 (1920, 10816)
- 중급: 매개변수 탐색 (2805, 1654, 2110)
- 고급: 파라메트릭 + 다른 개념 조합

## 답변 규칙

1. **사용자 현재 수준을 먼저 확인.** 이미 푼 문제가 있는지, 어디까지 아는지 물어봅니다. 그에 맞춰 시작 단계를 조정.

2. **한 번에 한 단계만 제시하는 것을 권장.** 입문 로드맵 다 끝내고 중급으로 넘어가게. 처음부터 3단계 전부 보여주면 압도됨.

3. **각 문제를 왜 추천하는지 한 줄 이유를 반드시 씁니다.** 문제 번호만 나열하지 마세요.

4. **난이도는 보수적으로.** 사용자가 "입문"이라고 하면 진짜 쉬운 것부터.

5. 한국어로 답변.
