풀이 코드 폴더(`boj/`, `codeforces/`, `atcoder/`) 와 분리된 **알고리즘 관련 지식** 저장소.
특정 알고리즘 자체의 증명이 아니라, 그 **배경 지식 / 기법 / 수학** 을 담는다.

## 서브 카테고리

- `theory/math/` — 수학·신호처리 배경 (DFT, 정수론, 조합론 등)
- `theory/technique/` — 알고리즘 설계·구현 기법 (재귀→반복 변환 등)

**Why:** 사용자가 "알고리즘 자체의 유도식보다는 관련 지식이니까 따로 분류하는 게 좋겠다" 고 명시.
풀이 중심 폴더와 메타 지식을 섞지 않기 위함.

**How to apply:**
- 새 이론/기법 문서는 위 카테고리에 맞춰 서브폴더 결정.
- 애매하면 사용자에게 분류 확인.
- 알고리즘 자체의 정당성 증명은 여기 말고 다른 곳(추후 결정, 현재 미정).
- 관련 문제·다른 문서와의 상호 링크는 적극 걸 것 (`../../boj/cpp/15576.cpp` 처럼 상대경로).
